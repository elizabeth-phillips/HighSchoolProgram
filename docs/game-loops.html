Adding Loops to our Game

A pattern emerged when we were drawing the grid on our game. We can stop repeating ourselves and solve the drawing of our game with a loop.

Our current drawing of the grid rows looks like the following:

canvas.create_line(50, 50, 550, 50, fill='gray', dash = (2,2)) #row1
canvas.create_line(50, 150, 550, 150, fill='gray', dash = (2,2)) #row2
canvas.create_line(50, 250, 550, 250, fill='gray', dash = (2,2)) #row3
canvas.create_line(50, 350, 550, 350, fill='gray', dash = (2,2)) #row4
canvas.create_line(50, 450, 550, 450, fill='gray', dash = (2,2)) #row5
canvas.create_line(50, 550, 550, 550, fill='gray', dash = (2,2)) #row6

From the above we can see that the x values remain the same, but the y value starts with 50 and increases by 100px each time. We can put this into a loop such as the following:

for n in range(50, 650, 100):
canvas.create_line(50, n, 550, n, fill='gray', dash = (2,2)) # create 6 rows

We took six lines of code and used logic to get it down to 2!
