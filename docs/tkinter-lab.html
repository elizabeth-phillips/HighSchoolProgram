TKINTER lab

Add a new file to your labs repl called tkinter_lab.py

Setup tkinter and your canvas with the following code

from tkinter import *
window = Tk()
window.title("Stick Figure Drawing")
canvas = Canvas(window, width=300, height=300)
canvas.pack()
window.mainloop(30)

With the following shape methods, draw a stick figure:

canvas.create_oval(x1, y1, x2, y2)
canvas.create_line(x1, y1, x2, y2)
canvas.create_arc(x1, y1, x2, y2)

where x1, y1, x2, y2 are your coordinates. The top right corner is your point of origin. As you move to the right, x value increases. As you move down, the y value increases.

Have fun with this!
