TKINTER

The tkinter package (“Tk interface”) is the standard Python interface to the Tk GUI toolkit. Both Tk and tkinter are available on most Unix platforms, as well as on Windows systems. (Tk itself is not part of Python; it is maintained at ActiveState.)

per the python documentation which can be found here.

Tkinter allows us to easily inteface with the GUI.

GUI stands for Graphical User Interface

Before we can use the tkinter library we're going to have to import it.

Add the following import statement to the top of your main.py file: from tkinter import * The * means import all the classes, functions, and objects from that library.

A library is a collection of code intented to be used by applications.

Next, we need to create instance of Tk from the tkinter library. To do that, we'll add the following code below our constants. window = Tk()

Let's give our GUI window a title. We can do that by calling the title method on our Tk instance. Add the following code at the bottom of your file window.title('Dots and Line Game')

Now we're ready to create a canvas.

We're going to use what's called a canvas to draw on the screen and present graphics to the user.

We can create the canvas and give it a size with the following:

canvas = Canvas(window, width=size_of_board, height=size_of_board) canvas.pack()

We're also going to want to put the window on a loop or its going to run our python code and quit. Add the following at the end of main.py window.mainloop(30)

Now the fun part! We're going to start drawing on our canvas! After the lab, to give you a change to draw some different shapes, we'll start drawing on our game!
